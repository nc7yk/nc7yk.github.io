西湖论剑2023-pwn复现

发表于2023-03-05|更新于2023-03-07
|阅读量:

Message Board

输入了长度为8的格式化字符串，可以使用%p和%24$p分别泄露栈地址和libc地址

溢出8字节，可以用于覆盖rbp和写入leave;ret，在上面泄露的栈地址处构造rop链，然后利用上面泄露的栈地址迁移执行即可

exp:

babycalc

漏洞点存在于执行循环后read可以通过写入buf内数据，控制循环变量 (i)，并由*(&v3 + i) = v0;实现栈上任意地址写一字节，一定几率可以写到rbp上，实现栈迁移

先通过z3算出结果（ps：存在一定小bug，可能是运算顺序决定了运算方式，第二个是正确的

rs = '\x13\x24\x35\x46\x37\x42\x11\xa1\x32\x83\xd4\x65\x76\xc7\x18\x03'

那么现在思路便是，在buf上布置rop链，并通过计算偏移利用*(&v3 + i) = v0;部分覆盖rbp到rop链上方执行

为了增加成功率，可以在rop上面写入多个ret一路滑到rop链
